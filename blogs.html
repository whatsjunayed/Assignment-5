Question-1 : What are the different ways to select on element in the DOM?

We can select DOM elements in different ways like using getElementsByTagName(), getElementById(), getElementsByClassName(), querySelector(), querySelectorAll().

Question-2 : What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is:

innerHTML: It returns the HTML tags and content of an element.

innerText: It returns only text content excluding HTML tags and influenced by CSS style.

textContent: It returns only text content excluding HTML tags and not affected by CSS style.

Question-3 : What is event delegation in the DOM?

Event delegation is a method in which an event listener is added to a parent element, and through it handles the events of the child elements under the parent element. It supports dynamic elements.

Question-4 : What is event bubbling in the DOM?

Event bubbling is a process where an event is first triggered on the event element and then gradually propagates to the parent elements until it reaches the root document.

Question-5 : How do you create, add, and remove elements using JavaScript?

⇒ To create a new element, document.createElement() is used
Example:
const createElem = document.createElement('div');

⇒ To add a new element, appendChild() is used
Example:
const parent = document.getElementById('parentElement');
parent.appendChild(newChild);

⇒ To remove an element, removeChild() or remove() is used
Example:
parent.removeChild(child);
or
child.remove();
